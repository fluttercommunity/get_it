get<T> method

T get <T>({String instanceName, dynamic param1 dynamic param2 })

retrieves or creates an instance of a registered type T depending on the registration function used for this type or based on a name. for factories you can pass up to 2 parameters param1,param2 they have to match the types given at registration with registerFactoryParam()
